## 🎯 如何运行游戏

### Windows 系统

```bash
# 第一次运行
install.bat      # 安装依赖
start.bat        # 启动游戏

# 或手动运行
pip install -r requirements.txt
python game.py
```

### 其他平台

```bash
# 安装依赖
pip install -r requirements.txt

# 运行游戏
python game.py
```
