### 修复内容汇总

#### UI 改进
1. ✅ 添加缺失的休息按钮
2. ✅ 为炼丹、任务、背包按钮添加"敬请期待"提示

#### 游戏逻辑修复
1. ✅ 探索功能添加生命值检查（<30 时提示，=0 时无法探索）
2. ✅ 战斗功能添加死亡检测和游戏结束处理
3. ✅ 探索受伤后添加死亡检测
4. ✅ 休息功能添加死亡检测
5. ✅ 实现完整的游戏结束机制（handleDeath）

#### 游戏结束机制
- 战斗中生命值归零 → 游戏结束
- 探索危险地点生命值归零 → 游戏结束
- 休息时检测已死亡 → 游戏结束
- 游戏结束时显示统计信息并清除存档
