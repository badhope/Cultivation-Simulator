# 修仙模拟器 - 华丽 GUI 版

## 🎮 游戏简介

这是一个全新重构的修仙模拟游戏，采用现代化的 GUI 界面，提供沉浸式的修仙体验。游戏在原有框架基础上进行了全面升级，包含：

- ✨ **华丽的图形界面**：使用 CustomTkinter 打造的现代化 UI
- 🎨 **动态动画效果**：流畅的过渡动画和粒子特效
- 🔊 **音效支持**：背景音乐和音效增强体验
- 📖 **宏大世界观**：完整的上古历史、天地法则体系
- 🎭 **丰富剧情**：多篇章主线剧情和蝴蝶效应系统
- 👥 **智能 NPC**：具有性格和社交关系的 NPC 群体
- 💰 **经济系统**：完整的市场、拍卖行、商会机制
- ⚔️ **修仙体系**：从凡人到渡劫的完整修炼路径

## 🚀 快速开始

### 方法一：运行启动脚本（推荐）

```bash
python start_gui_game.py
```

此脚本会自动检查并安装所需依赖。

### 方法二：直接运行主程序

```bash
python gui_main.py
```

### 方法三：使用批处理文件（Windows）

双击 `start_game.bat` 即可启动。

## 📦 系统要求

- **Python**: 3.8 或更高版本
- **操作系统**: Windows / macOS / Linux
- **内存**: 至少 2GB RAM
- **显示**: 支持 1200x800 分辨率

## 🔧 依赖包

主要依赖：
- `customtkinter` - 现代化 GUI 框架
- `pygame` - 音效和多媒体支持
- `pillow` - 图像处理

自动安装命令：
```bash
pip install customtkinter pygame pillow
```

或使用 requirements.txt:
```bash
pip install -r requirements.txt
```

## 🎯 游戏玩法

### 创建角色

1. 在启动界面点击"开始游戏"
2. 输入你的仙名
3. 分配初始属性点（共 20 点）：
   - **体质**：影响生命值和恢复速度
   - **灵根**：影响灵气吸收效率
   - **悟性**：影响学习和领悟速度
   - **机缘**：影响奇遇概率

### 主要功能

#### 🧘 修炼
- 提升修为值
- 达到 100 后可尝试突破境界
- 灵根越高，修炼效率越好

#### 🗺️ 探索
- 前往不同地点：青云山脉、幽冥谷、天机城等
- 发现资源和奇遇
- 遭遇妖兽或 NPC

#### 👥 社交
- 与其他修士交流
- 建立人际关系
- 可能触发合作或冲突事件

#### 🎒 背包
- 查看和管理资源
- 使用丹药和法器
- 整理获得的物品

#### 📜 任务
- 接受剧情任务
- 完成任务获得奖励
- 推动故事发展

#### 📚 功法
- 学习新功法
- 练习提升熟练度
- 搭配适合自己的修炼体系

### 境界划分

```
凡人 → 练气期 → 筑基期 → 金丹期 → 元婴期 → 
化神期 → 合体期 → 大乘期 → 渡劫期 → 飞升
```

每个境界都需要修为值达到 100 并成功突破。

## 🌟 特色系统

### 1. 世界观系统
- **上古历史**：混沌初开、洪荒时代、封神时代等五个纪元
- **传奇人物**：盘古、鸿钧道人、通天教主等神话存在
- **天地法则**：大道、阴阳、五行、因果等法则可感悟
- **灵气演化**：不同地区灵气质量和浓度各异

### 2. 剧情系统
- **四大篇章**：凡人崛起、名扬一方、开宗立派、问鼎巅峰
- **蝴蝶效应**：你的选择会影响后续剧情和世界走向
- **多结局**：根据行为和成就解锁不同结局

### 3. 社交系统
- **NPC 性格**：8 种性格类型，影响对话和行为
- **人际关系**：好感度系统，从陌生到挚友
- **结拜系统**：与志同道合者结为兄弟
- **婚姻系统**：寻找道侣，双修共进

### 4. 经济系统
- **灵石货币**：下品、中品、上品、极品四种灵石
- **市场波动**：供需关系影响价格
- **拍卖行**：竞拍珍稀物品
- **黑市**：购买禁术和来路不明的宝物
- **商会**：加入或创建商会，经营贸易

### 5. AI 引导
- **智能提示**：根据状态提供建议
- **日常任务**：每日签到和指导
- **情感互动**：AI 引导员的情感化回应

## 💡 游戏技巧

1. **合理分配属性**：初期建议平均分配或侧重灵根

4. **市场投资**：低买高卖赚取灵石

### 高手策略

1. **蝴蝶效应利用**：通过选择引导有利剧情
2. **经济运作**：商会贸易获取巨额利润
3. **双修之道**：寻找合适道侣共同提升
4. **法则感悟**：后期注重法则理解
5. **多周目**：尝试不同路线和结局

## 🐛 常见问题

### Q: 游戏无法启动
A: 检查 Python 版本是否≥3.8，确保已安装所有依赖包。

### Q: 界面显示异常
A: 尝试调整屏幕分辨率至 1200x800 以上，更新显卡驱动。

### Q: 没有声音
A: 检查系统音量，确认 pygame 正确安装。游戏支持静音模式。

### Q: 游戏卡顿
A: 关闭其他占用资源的程序，降低窗口大小。

### Q: 存档丢失
A: 存档位于 `saves/` 目录，定期备份重要存档。
